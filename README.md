# Aurora

Aurora is an initial repository scaffold for organizing documentation, applications, services, shared packages, AI assets, infrastructure, scripts, tests, and tooling.

## Structure

- docs/ for architecture, API, database, development, roadmap, and decisions
- apps/ for web, desktop, mobile, and admin experiences
- services/ for domain-oriented backend services
- packages/ for shared libraries and configuration
- ai/ for AI-related assets and workflows
- infrastructure/ for platform and deployment assets
- scripts/, tests/, and tools/ for operational support

## Status

This repository currently contains the requested directory structure and foundational documentation files.
