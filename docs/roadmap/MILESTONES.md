# Milestones

## MVP
The MVP milestone focuses on delivering the core product experience with essential user flows, reliable infrastructure, and a minimal set of features that prove the platform concept.

## Beta
The Beta milestone expands the platform with broader feature coverage, improved stability, early user feedback loops, and stronger performance across the main product experiences.

## Public Release
The Public Release milestone marks the point where the product is ready for wider adoption, with polished onboarding, support readiness, and confidence in security and reliability.

## Aurora v2
Aurora v2 represents the next generation of the platform, introducing deeper product capabilities, better scalability, and a more refined experience based on lessons learned from earlier releases.

## Aurora AI Model
The Aurora AI Model milestone focuses on advancing the underlying intelligence layer, improving model quality, and integrating richer AI experiences into the platform.
